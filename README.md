# Конфигурационное управление. Текущий контроль. Домашнее задание. 
Разработать инструмент командной строки для учебного конфигурационного языка, 
синтаксис которого приведен далее. Этот инструмент преобразует текст из входного формата в выходной.
Синтаксические ошибки выявляются с выдачей сообщений. Входной текст на учебном конфигурационном языке
принимается из стандартного ввода. Выходной текст на языке yaml попадает в файл, путь к которому задан
ключом командной строки.

## 1. Запуск
Для запуска можно просто открыть консоль в папке репозитория, и вызвать команду

	py main.py

Или запустить файл run.bat, исполняющий вышеуказанную команду.
Для запуска так же доступен аргумент, -f или же -file_name, которое отвечает за имя выходного файла. 
По-умолчанию данный аргумент равен 'data'.
Для завершения работы программы нужно отправить пустую строку.

## 2. Переменные конфигурационного языка
Объявление переменной: Для объявления переменной необходимо указать слово global, имя переменной, знак равно и содержимое переменной.
Имя переменной: Содержит буквы латинского алфавита [a-z] а так же [A-Z], и цифры [0-9]
Содержимое переменной: Переменная может содежрать в себе 3 типа.
  1. Строка, представляет из себя набор любых символов, закрытый одиночными кавычками ['] с двух сторон
  2. Число, представляет из себя экспоненциальную запись числа, пример: 1.05e+2 , что переводится в 105
  3. Массив. Может содержать в себе все предыдущие типы, а так же другие массивы. Представляет из себя переменную,
	   или переменные, разделённые запятыми, закрытые квадратными скобками с двух сторон. Пример: [1.0e0, 'string']

## 3. Константные выражения
В качестве содержимого переменной можно объявить константное выражение, в виде @{+ имя_переменной число},
где + - знак константного выражения (доступны следующие выражения: +, -, /, *, min), имя_переменной - первый аргумент получает значение от другой переменной,
число - просто число. Пример реализации: 
	global var1=1.0e0
	global var2=@{+ var1 2.0e0}
Результатом выполнения этих строк будет то, что значение var2 будет равно 3.0e0.

## 4. Ограничения
Константное выражение не может содержать название переменной, содержимое которой оно описывает, пример: global test=@{+ test 1.0e0}
